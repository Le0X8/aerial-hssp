# Aerial Sources Downloader

This script...
- gets the URL of the sources
- downloads the source
- untars the source
- converts the binary string files into JSON
- packs the source and the strings into a [HSSP Archive](https://github.com/Le0X8/HSSP)


# Usage

1. Clone the repo
2. Run `node . <VERSION>` (replace the \<VERSION\> with an integer above 9, this is the tvOS version).
3. Do whatever you want with the newly created _resources.hssp_ file in the project folder.
